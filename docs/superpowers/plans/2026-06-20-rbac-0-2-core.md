# RBAC 0.2 Core Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the additive 0.2.0 core API surface from `docs/spec-0.2.0.md` without breaking existing 0.1.x behavior.

**Architecture:** Add focused helpers and interfaces around the existing service, guard, storage, audit, and testing boundaries. Keep current string permissions, defaults, and Prisma schema compatible. New safety behavior is opt-in through helpers/options.

**Tech Stack:** TypeScript, NestJS, Vitest, tsup, existing in-memory storage and package export conventions.
